---
description: Create a commit and push to remote
---

1. Run the following command to stage all changes:
   ```bash
   git add .
   ```
2. Run the following command to commit the changes. Note: You should update the commit message to be descriptive of your changes.
   ```bash
   git commit -m "update code"
   ```
// turbo
3. Run the following command to push to remote:
   ```bash
   git push
   ```
